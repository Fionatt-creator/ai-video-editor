# AI 视频剪辑助手 - MVP 测试版本

## 🚀 测试地址

GitHub Pages: https://fionatt-creator.github.io/ai-video-editor

## ✨ 完整功能流程

### 1. 上传素材
- 支持拖拽上传视频文件
- 显示文件信息和格式支持
- 最近项目快速访问

### 2. AI 智能分析
- 实时进度显示（圆形进度条）
- 分步骤显示处理状态：
  - ✅ 上传完成
  - 🔄 提取音频
  - 🔄 语音识别
  - 🔄 分析静音片段
  - 🔄 识别高光时刻
  - 🔄 生成时间线
- 预计剩余时间计算

### 3. 时间轴精修（核心功能）
- **可视化时间轴**：彩色区分不同类型的片段
  - 🟣 紫色：高光片段（高置信度）
  - 🔵 蓝色：保留片段
  - ⚫ 灰色：删除片段
- **片段管理**：
  - 点击选中片段
  - 切换片段类型（高光/保留/删除）
  - 一键删除/恢复片段
  - 显示 AI 评分（0-100分）
- **播放控制**：
  - 播放/暂停
  - 前进/后退
  - 时间轴游标
- **统计信息**：
  - 预计节省时间
  - 总时长对比

### 4. 导出设置
- 多格式支持：
  - 📱 竖屏版（9:16）- 抖音/视频号
  - 💻 横屏版（16:9）- B站/YouTube
  - ⬜ 方形版（1:1）- 小红书/朋友圈
- 分辨率选择（720p/1080p）
- 字幕开关
- 导出进度显示
- 下载成片

## 🎯 测试要点

1. **上传流程**：尝试拖拽上传视频文件
2. **AI 分析**：观察进度条和步骤状态变化
3. **时间轴编辑**：
   - 点击不同片段查看详情
   - 尝试删除和恢复片段
   - 切换片段类型
4. **导出功能**：选择不同格式导出

## 🐛 已知限制

- 视频处理是模拟的（非真实 AI 处理）
- 预览区域为占位符（无真实视频播放）
- 导出为模拟下载
- 仅支持单个视频处理
