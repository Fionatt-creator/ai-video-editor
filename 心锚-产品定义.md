# 人生幸福指南 - 产品定义文档

## 产品四问

### 1. 为谁做？（目标用户）
**核心用户**：25-40岁、受过良好教育、处于"意义真空"状态的都市人群
- 物质需求基本满足，但精神层面感到空虚
- 经常思考"我为什么要活着"但找不到答案
- 尝试过阅读、冥想、咨询等方式但效果有限
- 对AI工具持开放态度，愿意尝试新方法

**用户画像**：
- 小李，28岁，互联网大厂产品经理，年薪50w+，但每天加班后感到极度空虚
- 小王，32岁，已婚无孩，工作稳定，但总觉得"这不是我想要的生活"
- 小张，26岁，刚毕业2年，换了3份工作，仍不知道自己喜欢什么

### 2. 解决什么痛点？（核心问题）
**一级痛点**：存在性空虚 + 方向迷失
- 用户原话："我不知道自己为什么而活"
- 用户原话："道理都懂，但就是做不到"

**二级痛点**：
- 自我探索工具太抽象（哲学书看不懂、测试做完没行动）
- 现有方案太"说教"（感觉在被教育，产生抗拒）
- 缺乏个性化（通用的建议不适合自己的情况）

### 3. 提供什么价值？（核心价值主张）
**一句话价值**：通过AI引导式对话，帮用户在2-4周内找到属于自己的"人生意义锚点"

**价值分层**：
1. **即时价值**：每次对话后感到被理解、情绪被接纳
2. **短期价值**：完成自我探索旅程，产出专属的"人生意义地图"
3. **长期价值**：获得持续的意义感来源，建立自我觉察习惯

### 4. 为什么选你？（差异化）

| 竞品/替代方案 | 他们的局限 | 我们的优势 |
|-------------|----------|-----------|
| 心理学书籍 | 太抽象，难落地 | 互动式引导，一步步来 |
| MBTI等测试 | 贴标签，没行动 | 动态生成，持续进化 |
| 心理咨询 | 贵、门槛高、频次低 | 随时可用、性价比高 |
| 自律打卡APP | 治标不治本 | 从意义源头解决问题 |
| 其他AI助手 | 通用对话，缺乏结构 | 专门的"意义发现"框架 |

**核心差异化**：
- **引导式而非说教式**：AI不告诉用户"应该"怎么做，而是通过提问帮助用户自己发现
- **结构化 + 个性化**：基于经过验证的心理学框架，但为每个用户生成独特的内容
- **行动导向**：不只停留在"想明白"，而是转化为可执行的行动

## 产品定义

### 产品名称
**「心锚」**（英文名：LifeAnchor）

**命名逻辑**：
- 锚：让人生不再漂泊，找到稳固的支点
- 心锚：发自内心的、属于自己的锚点
- 双关："新锚" → 新的人生锚点

### 一句话定位
> 「心锚」是你的AI意义发现伴侣，通过结构化自我探索，帮你找到属于自己的人生方向。

### 产品形态
- 移动端优先（随时随地使用）
- AI对话式交互（自然、私密）
- 辅助Web端查看"人生意义地图"可视化

## MVP 功能定义（仅3个核心功能）

### 功能1：意义发现对话（核心）
**功能描述**：
AI通过多轮引导式对话，帮助用户探索价值观、热情、人生目标。

**关键特性**：
- 采用「ACT接纳承诺疗法」+「意义疗法」框架
- 每轮对话聚焦一个维度（价值观/热情/目标/行动）
- AI不输出建议，只输出问题和反馈
- 对话历史形成"探索档案"

**用户旅程示例**：
1. 用户："我不知道自己想要什么"
2. AI："我们先不急着找答案。回想一下，最近一次让你感到'这才是我'的时刻是什么时候？"
3. 用户回答...
4. AI追问、澄清、反馈
5. 最终输出：用户的核心价值观列表

### 功能2：人生意义地图（可视化）
**功能描述**：
将用户的探索结果可视化为一张"人生意义地图"，包含价值观、热情、目标、行动四个维度。

**关键特性**：
- 自动生成可视化地图
- 支持用户手动调整
- 可导出分享（可选匿名）
- 随探索深入动态更新

**地图结构**：
```
         [核心价值观]
              |
    [热情领域]——[人生目标]——[日常行动]
              |
         [支持系统]
```

### 功能3：意义微行动（行动化）
**功能描述**：
基于用户的人生意义地图，AI每日推荐一个"微行动"，帮助用户将意义感融入日常生活。

**关键特性**：
- 每日1个微行动（5-15分钟可完成）
- 与用户地图强关联
- 完成后记录反思
- 累积形成"意义习惯"

**示例微行动**：
- "今天找一个安静的时刻，写下3件你感激的事情"
- "做一件与你'创造力'价值观相符的小事"
- "给那个让你感到被理解的人发条消息"

## MVP 完成标准

1. **功能1完成**：用户能完成至少一次完整的意义发现对话，并获得有意义的输出
2. **功能2完成**：用户能看到自己的人生意义地图可视化
3. **功能3完成**：用户能收到并完成为期一周的每日微行动
4. **留存指标**：7日留存率 > 30%
5. **满意度**：完成首次对话的用户，NPS > 40
