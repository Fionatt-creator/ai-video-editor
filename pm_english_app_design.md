# PM 场景英语 - 移动端产品设计方案

## 一、产品定位

**产品名称**：PM English 场景训练
**Slogan**：每天一个场景，练就职场英语
**目标用户**：
- 初级-中级产品经理（1-5年经验）
- 有英语学习需求的互联网从业者
- 准备海外/外企面试的职场人

**核心价值**：将抽象的产品经理职责转化为可观看、可模仿、可练习的沉浸式场景视频

## 三、技术实现方案

### 3.1 AI视频生成

**方案A：数字人方案（推荐）**
- 使用 HeyGen/D-ID 等数字人 API
- 优势：口型精准、专业感强
- 成本：约 $2-3/分钟

**方案B：动画方案**
- 使用卡通风格角色动画
- 优势：成本低、风格轻松
- 成本：约 $0.5/分钟

**预生成策略**：
- 首次用户访问时后台生成
- 热门场景预生成缓存
- 用户也可选择"文字+音频"模式（即时加载）

### 3.2 技术架构

```
前端（React Native / Flutter）
     ↓
后端 API（Node.js/Python）
     ↓
├─ 场景数据管理（飞书 Bitable）
├─ AI视频生成服务（HeyGen API）
├─ 字幕处理服务
└─ 用户学习记录（数据库）
     ↓
视频存储（阿里云OSS/AWS S3）
CDN 加速
```

### 3.3 数据库设计

**场景表 (scenes)**
```sql
- id: 场景ID
- category: 职责分类
- title_cn: 中文标题
- title_en: 英文标题
- dialogue_en: 英文对话
- dialogue_cn: 中文翻译
- keywords: 高频词汇（JSON）
- phrases: 核心词组（JSON）
- video_url: 视频链接（生成后填充）
- duration: 视频时长
```

**用户学习记录 (user_progress)**
```sql
- user_id: 用户ID
- scene_id: 场景ID
- status: 学习状态（未开始/进行中/已完成）
- play_count: 播放次数
- last_position: 上次播放位置
- completed_at: 完成时间
```

## 四、功能模块详情

### 4.1 场景浏览器

**筛选维度**：
- 按职责分类（11个类别）
- 按难度（初级/中级/高级）
- 按场景类型（会议/1对1/邮件）
- 按时长（<2分钟 / 2-5分钟 / >5分钟）

**搜索功能**：
- 关键词搜索（英文/中文）
- 模糊匹配对话内容

### 4.2 视频播放器

**核心功能**：
- 倍速播放（0.5x - 2x）
- 字幕开关
- 单句循环
- AB点复读
- 后台播放（音频模式）

**字幕交互**：
- 点击单词显示释义
- 收藏到生词本
- 一键复制对话

### 4.3 练习模式

**模式1：跟读练习**
- 系统播放一句
- 用户录音跟读
- AI评分（发音准确度）

**模式2：角色扮演**
- 选择角色（PM1/PM2/PM Lead）
- 隐藏该角色台词
- 用户配音演绎
- 与原版对比

**模式3：听力填空**
- 隐藏关键词
- 听视频填空
- 检验听力理解

### 4.4 生词本

- 自动收藏点击过的词汇
- 按场景分类
- 复习提醒（艾宾浩斯遗忘曲线）
- 导出功能（PDF/Anki）

### 4.5 学习统计

- 学习时长统计
- 场景完成进度
- 词汇掌握量
- 连续学习天数
