# AI 视频剪辑助手 - MVP 产品需求文档 (PRD)

**版本**: v1.0  
**日期**: 2026-03-14  
**目标用户**: 影视解说博主、知识付费讲师、MCN 机构剪辑师

---

## 一、产品定位

### 1.1 核心价值主张

> **"把 3 小时的机械劳动压缩到 30 分钟，让创作者回归创意"**

### 1.2 产品定位

**"AI 智能粗剪 + 人工精修" 的混合模式**

- **自动化处理 80%** 的重复性机械工作（素材筛选、静音删除、初版剪辑）
- **保留 20% 人工控制** 用于创意精修（画面调整、节奏微调、个性化表达）

### 1.3 目标场景

| 场景 | 描述 | 目标用户 |
|------|------|----------|
| 影视解说 | 长视频素材 → AI 粗剪 → 人工精修 → 多平台发布 | 影视解说博主 |
| 知识付费 | 课程录像 → 智能分段 → 字幕生成 → 精华片段提取 | 讲师/教育机构 |
| 矩阵运营 | 批量素材 → 自动高光识别 → 多版本导出 | MCN/运营团队 |

## 二、MVP 功能范围

### 2.1 核心功能（Must Have）

#### F1: AI 智能粗剪
- **功能描述**: 自动识别高光片段、删除静音/无效片段、生成初版时间线
- **用户价值**: 解决"剪辑效率低"痛点，节省 70%+ 粗剪时间
- **关键指标**: 处理 10 分钟素材 < 2 分钟，保留片段准确率 > 85%

#### F2: AI 字幕生成
- **功能描述**: 语音识别自动生成字幕，支持时间轴微调
- **用户价值**: 解决"字幕制作慢"痛点，准确率 > 95%
- **关键指标**: 中文识别准确率 > 95%，生成速度 1:0.5（素材时长:处理时间）

#### F3: 一键多平台适配
- **功能描述**: 自动生成横屏(16:9)、竖屏(9:16)、方形(1:1)三个版本
- **用户价值**: 解决"多平台适配烦"痛点，一次剪辑多平台发布
- **关键指标**: 横转竖主体追踪准确率 > 90%

#### F4: 人工精修界面
- **功能描述**: 可视化时间轴，支持片段增删、顺序调整、转场添加
- **用户价值**: 保留创作控制，修正 AI 误判
- **关键指标**: 精修操作响应 < 100ms

### 2.2 延后功能（Nice to Have / Phase 2）

| 功能 | 延后原因 | 预计上线 |
|------|----------|----------|
| AI 脚本生成 | 依赖大模型，成本高，可先集成文案模板 | Phase 2 |
| 数字人/AI 配音 | 非核心痛点，可作为增值功能 | Phase 2 |
| 云端素材库 | 存储成本高，先做本地版验证 | Phase 2 |
| 一键多平台发布 | 需对接各平台 API，合规复杂 | Phase 2 |
| 团队协作/审片 | 面向团队用户，先验证个人需求 | Phase 3 |

## 三、用户旅程流程

### 3.1 主流程：从素材到成片

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   上传素材   │ → │  AI 粗剪    │ → │  人工精修   │ → │  字幕/导出  │ → │  下载成片   │
│  (视频文件)  │    │ (自动处理)   │    │ (可视化编辑) │    │ (一键导出)   │    │ (多平台版本) │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      ↓                  ↓                  ↓                  ↓
   支持格式:           自动生成:          可调整:           自动生成:
   MP4/MOV/AVI        • 高光片段          • 片段顺序         • 字幕
   最大 2GB           • 删除静音          • 片段边界         • 多尺寸版本
                      • 初版时间线        • 转场效果         • 封面截图
```

### 3.2 详细流程步骤

**Step 1: 上传素材** (2分钟)
- 拖拽或选择本地视频文件
- 显示上传进度
- 自动检测视频信息（时长、分辨率、音频轨道）

**Step 2: AI 粗剪配置** (1分钟)
- 选择内容类型：影视解说 / 知识课程 / Vlog 日常
- 设置目标时长：1分钟 / 3分钟 / 5分钟 / 原时长
- 选择输出格式：横屏 / 竖屏 / 全部

**Step 3: AI 处理** (2-5分钟，后台运行)
- 语音识别转文字
- AI 分析高光片段
- 自动删除静音/语气词
- 生成初版时间线

**Step 4: 人工精修** (10-20分钟)
- 可视化时间轴浏览
- 拖拽调整片段顺序
- 裁剪片段起止点
- 添加转场效果
- 预览成片效果

**Step 5: 字幕与导出** (2分钟)
- 一键生成字幕，可编辑校对
- 选择导出格式：MP4 / MOV
- 选择分辨率：720p / 1080p / 4K
- 批量导出多平台版本

## 四、功能详细设计

### 4.1 AI 智能粗剪模块

#### 输入
- 视频文件（支持 MP4, MOV, AVI, MKV）
- 配置参数：内容类型、目标时长、保留比例

#### 处理逻辑
1. **语音转文字**: 使用 Whisper 模型生成带时间轴的字幕
2. **语义分析**: 识别关键词、高潮点、情感强度变化
3. **画面分析**: 检测场景切换、人物出现、关键动作
4. **静音检测**: 识别无语音段落、语气词（嗯/啊/那个）
5. **片段评分**: 综合多维度评分，排序选出高光片段
6. **时间线生成**: 按目标时长组合片段，保持叙事连贯

#### 输出
- 初版时间线（片段列表 + 时间码）
- 每片段的 AI 评分和推荐理由
- 被删除片段列表（可恢复）

### 4.2 人工精修界面

#### 界面布局
```
┌─────────────────────────────────────────────────────────────┐
│  [预览窗口]                           [片段列表]            │
│  ┌───────────────┐                    ┌──────────────┐     │
│  │               │                    │ 片段1 [95分] │     │
│  │    视频预览    │                    │ 片段2 [88分] │     │
│  │               │                    │ 片段3 [82分] │     │
│  │               │                    │ ...          │     │
│  └───────────────┘                    └──────────────┘     │
├─────────────────────────────────────────────────────────────┤
│  [时间轴]                                                   │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  ████    ██████      ████      ████████              │  │
│  │ 片段1   片段2       片段3       片段4                │  │
│  └──────────────────────────────────────────────────────┘  │
├─────────────────────────────────────────────────────────────┤
│  [工具栏]  [撤销] [恢复] [分割] [删除] [转场] [字幕]        │
└─────────────────────────────────────────────────────────────┘
```

#### 核心交互
- **片段拖拽**: 调整片段顺序
- **边缘拉伸**: 调整片段起止时间
- **右键菜单**: 删除/复制/分割片段
- **快捷键**: Space(播放), Delete(删除), Ctrl+Z(撤销)

### 4.3 多平台适配

#### 自动构图逻辑
- **横屏(16:9)**: 保持原画幅
- **竖屏(9:16)**: 
  - 智能追踪人脸/主体位置
  - 自动调整画面裁剪区域
  - 保持主体在画面中心 1/3 区域
- **方形(1:1)**: 
  - 取画面中心区域
  - 智能避让人物被裁剪

#### 字幕适配
- 横屏: 底部居中，字号 48px
- 竖屏: 底部居中，字号 36px，避免被平台 UI 遮挡
- 方形: 底部居中，字号 40px

## 五、技术实现要点

### 5.1 技术栈选型

| 模块 | 技术方案 | 选型理由 |
|------|----------|----------|
| 前端 | React + TypeScript | 生态成熟，组件丰富 |
| 视频编辑 | FFmpeg.wasm + Canvas | 浏览器端处理，无需上传 |
| AI 语音 | Whisper (OpenAI) | 中文准确率高，开源免费 |
| AI 视频分析 | 自研模型 + OpenCV | 针对高光识别场景优化 |
| 后端 | Node.js + Express | 快速开发，生态丰富 |
| 存储 | 本地 IndexedDB | MVP 阶段优先本地处理 |

### 5.2 关键技术难点

1. **浏览器端视频处理**
   - 使用 FFmpeg.wasm 实现本地转码
   - 大文件分片处理，避免内存溢出
   - Web Worker 多线程处理

2. **AI 模型部署**
   - Whisper 模型量化压缩，适配浏览器
   - 高光识别模型轻量化为 ONNX 格式
   - 考虑 WebGPU 加速推理

3. **实时预览性能**
   - 使用 Media Source Extensions (MSE)
   - 片段预览生成低分辨率缩略图
   - 最终导出时再合成高分辨率
