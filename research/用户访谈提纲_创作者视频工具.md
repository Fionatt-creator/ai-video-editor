# 创作者视频工具用户访谈提纲

**访谈目标**：验证痛点优先级、挖掘潜在需求、测试付费意愿
**访谈对象**：影视解说博主 / 知识付费讲师 / MCN剪辑师
**访谈时长**：30-40分钟
**记录方式**：录音（需征得同意）+ 访谈笔记

---

## 一、开场破冰（3分钟）

1. 简单自我介绍，说明访谈目的
2. 确认对方身份：
   - "您主要做哪类内容？做多久了？"
   - "团队规模？（个人/小团队/MCN）"
   - "主要发布在哪些平台？"

---

## 二、当前工作流程（5分钟）

**目的**：了解真实工作流，找到耗时环节

1. "从拿到素材到最终发布，您的标准流程是什么？能一步步描述吗？"
   - 提示：素材整理 → 粗剪 → 精剪 → 字幕 → 封面 → 导出 → 发布

2. "每个环节大概花多长时间？"
   - 请对方估算：素材筛选（ ）分钟、剪辑（ ）分钟、字幕（ ）分钟、导出（ ）分钟

3. "一周大概产出多少条内容？每条平均时长？"

---

## 三、痛点挖掘（10分钟）

**目的**：量化痛点，验证调研结论

### 3.1 剪辑效率
4. "剪辑过程中，最让您烦躁的三个环节是什么？"
   - 追问："如果只能解决一个，您选哪个？为什么？"

5. "有没有因为剪辑太麻烦而放弃发布的内容？大概占比多少？"

6. "您现在怎么找高光片段？（手动看 / AI识别 / 其他）"
   - 追问："如果AI能自动挑出精彩片段，您觉得能省多少时间？"

### 3.2 字幕制作
7. "您的字幕是怎么做的？（手动打 / AI生成后修改 / 外包）"
   - 追问："准确率如何？修改要花多久？"

8. "有没有因为字幕问题被观众反馈过？"

### 3.3 多平台适配
9. "您会在多个平台发同样的内容吗？"
   - 追问："需要为不同平台调整格式吗？花多久？"

10. "横屏改竖屏怎么做的？（手动裁 / 工具自动 / 不发竖屏）"

### 3.4 工具使用
11. "您现在主要用什么剪辑工具？用了多久？"
    - 追问："给这个工具打几分（1-10）？扣分点在哪？"

12. "有没有试过其他工具？为什么没坚持用？"

---

## 四、需求验证（8分钟）

**目的**：测试产品概念，验证付费意愿

### 4.1 概念测试
13. "如果有这样一个工具：
    - 上传素材后，AI自动识别高光片段、删除静音和废话
    - 一键生成带字幕的初版时间线
    - 您可以微调后导出
    - 还能一键生成横竖屏多版本
    
    您会觉得有用吗？"
    - 追问："哪个功能对您最值钱？"

14. "这个工具如果能把您的剪辑时间从3小时压缩到30分钟，您愿意付多少钱？"
    - 选项：免费 / 50元/月 / 100元/月 / 200元/月 / 按条计费（ ）元/条

### 4.2 竞品对比
15. "您用过剪映的AI功能吗？（智能解说粗剪、图文成片）体验如何？"

16. "听说过 Wisecut、Firecut 这类AI剪辑工具吗？试过吗？"

---

## 五、开放收尾（5分钟）

17. "除了我们聊的这些，您在内容创作上还有什么烦恼？"

18. "如果让您设计一个理想的剪辑工具，它必须有什么功能？"

19. "您愿意在产品内测时体验并给反馈吗？"
    - 留联系方式

## 访谈执行 checklist

- [ ] 提前发送访谈提纲给对方
- [ ] 准备录音设备/软件，征得同意
- [ ] 准备好纸笔记录关键数据
- [ ] 访谈结束后24小时内整理纪要
- [ ] 将核心发现同步到飞书文档

---

**访谈技巧**：
1. 多问"为什么"，少问"是不是"
2. 让对方讲故事，而不是给评价
3. 记录具体数字（时间、频率、占比）
4. 留意对方的情绪变化（提到哪个环节时最激动/无奈）
